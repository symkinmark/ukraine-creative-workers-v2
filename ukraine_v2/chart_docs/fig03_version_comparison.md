# Fig 03 — Version Comparison: V1 vs V2.1

**File:** `charts/fig03_version_comparison.png`

## What this chart shows
How our main finding changed between the first version of the study (V1, n=415) and the current expanded version (V2.1, n=6,106). Uses the same two-group framing — Migrated vs Stayed in Soviet Sphere — to make the comparison fair.

## Key finding
| | V1 (n=415) | V2.1 (n=6,106) |
|---|---|---|
| Migrated LE | 72.0 yrs | 75.9 yrs |
| Stayed LE | 63.0 yrs | 70.5 yrs |
| Gap | +9.0 yrs | +5.4 yrs |

The gap narrowed from 9.0 to 5.4 years as the dataset grew 15x. This is expected and actually strengthens the paper: V1 was small and likely over-estimated the gap (selection bias — only the most famous / well-documented people). V2.1 is more representative and still shows a significant gap.

## What to look for
- The gap narrowed but did **not** disappear — the finding is robust
- Both migrated and stayed values increased in V2.1 (more complete data raises all estimates)
- The note at the bottom explains the "Stayed" group definition (non-migrated + internal transfer + deported combined)

## Group definition note
"Stayed in Soviet Sphere" = non_migrated + internal_transfer + deported combined. This collapses the four-group model back to two groups for comparison with V1, which didn't have the four-way split. The four-group version is in fig01 and is the preferred framing for V2.1.
